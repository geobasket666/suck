# Telegram Tutor Reminder Bot

Личный Telegram-бот для репетитора: хранит учеников, разовые и еженедельные занятия, показывает расписание и присылает напоминания.

## Быстрый запуск

1. Создай бота в Telegram через `@BotFather` и получи токен.
2. Создай виртуальное окружение и установи зависимости:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

3. Скопируй пример настроек:

```bash
cp .env.example .env
```

4. Открой `.env` и вставь токен:

```env
BOT_TOKEN=твой_токен_от_BotFather
TIMEZONE=Europe/Moscow
DATABASE_PATH=tutor_bot.sqlite3
PROXY_URL=
```

`TEACHER_CHAT_ID` можно оставить пустым. При первом `/start` бот запомнит твой чат и будет присылать напоминания туда.

Если с этого Mac не открывается `api.telegram.org`, включи VPN или укажи прокси:

```env
PROXY_URL=socks5://127.0.0.1:1080
```

Для HTTP-прокси формат такой:

```env
PROXY_URL=http://127.0.0.1:8080
```

5. Запусти бота:

```bash
python bot.py
```

## Команды

После `/start` бот покажет меню с кнопками:

- `Новое занятие`
- `Новый ученик`
- `Быстрый ввод`
- `Изменить расписание`
- `Сегодня`
- `Неделя`
- `Ближайшие`

Добавление занятия теперь идет пошагово: выбираешь ученика кнопкой, вводишь дату и время, выбираешь длительность, выбираешь разовое или еженедельное занятие, добавляешь заметку или пропускаешь ее.

Изменение расписания тоже идет через кнопки: выбираешь ученика, выбираешь его занятие, затем меняешь дату/время, длительность, тип повтора, заметку или удаляешь занятие.

Быстрый ввод добавляет еженедельные занятия одной строкой:

```text
Захар ср 20:30
Никита ср 19:30, чт 19:00
София среда 17:00 90
```

Если длительность не указана, бот ставит 60 минут.

```text
/start
/help
/cancel
/new_lesson
/q Захар ср 20:30
/edit_schedule
/add_student Имя | Предмет | Заметки
/students
/add_lesson ID_ученика | 2026-04-29 18:30 | 60 | once | заметка
/add_lesson ID_ученика | 2026-04-30 17:00 | 90 | weekly | математика
/today
/week
/lessons
/delete_lesson ID_занятия
```

Формат занятия:

```text
/add_lesson ID_ученика | YYYY-MM-DD HH:MM | длительность_в_минутах | once/weekly | заметка
```

Напоминание приходит за 60 минут до занятия. Для еженедельных занятий бот после напоминания автоматически перенесет следующее занятие на неделю вперед.
